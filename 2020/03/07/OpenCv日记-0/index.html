图像载入显示:

  • 图像载入: imread()函数
    函数原型:

filename:是我们需要填入的图片路径
flag:指定颜色类型，默认为1，指的是3通道的彩色图像，当值为0时则为灰度图像。

  • 图像显示:imshow()函数
    函数原型:

winname:填需要显示的窗口标识名称.
mat:InputArray类型的mat，填需要显示的图像

  • 创建窗口:namedWindow()函数
    原型:

参数1: 窗口名称
参数2: WINDOW_NORMAL是用户可以改变窗口大小，WINDOW_AUTOSIZE不能改变大小自适应,WINDOW_OPENGL窗口会支持OpenGL

  • 输出图像到文件:imwrite()函数
    函数原型：

参数1:文件名
参数2:Mat类型数据
参数3:特定格式参数编码

滑动条的创建和使用

  • 创建滑动条:createTrackbar()函数
    原型:

参数1:轨迹条的名字
参数2：表示轨迹条会依附到哪个窗口上，应该用namedWindow()函数创建某个窗口
参数3：表示滑块的位置，在初始时应是该变量当前的值
参数4：最大值位置，最小值始终为0
参数5：这是一个回调函数，很有用
参数6：不介绍

HighGUI图像的初步基本函数总结

imread 函数

读取文件中的图片到OpenCV中

imshow 函数

指定一个窗口显示一幅图像

namedWindow 函数

创建一个窗口

imwrite 函数

输出图像到文件

createTrackbar 函数

创建一个可以用于调整数值的轨迹条

getTrackbarPos 函数

获取轨迹条的当前位置

setMouseCallback 函数

指定的窗口设置鼠标的回调函数
